Modes Overview

Modes are for charms that have different separate functions, for example a charm that has an instant version or a scene long version

Set the mode using the + button and then the dialog will allow you enter information for things like duration and cost

There are trigger options that let you check for certain modes, allowing for effects tied to certain charm configurations

Modes are set using the button in the charms item listing in the character sheet.

Note: Set all (Non-trigger/bonuses) fields on your charm before setting any modes
